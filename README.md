**Online Retail Performance Dashboard**

An interactive Power BI dashboard designed to analyze and track key performance indicators (KPIs) for an online retail business. This project transforms raw transactional data into actionable insights, helping stakeholders understand sales trends, customer behavior, and product performance.


**Project Overview**

This repository contains the source data and the Power BI implementation for a retail analytics dashboard. The analysis focuses on uncovering sales patterns, identifying top-performing products, and analyzing regional revenue distribution.


**Key Features**

_-Sales Performance Tracking:_ Real-time visibility into total revenue, average order value, and sales volumes.

_-Customer Insights:_ Breakdown of customer purchasing patterns and regional demographic performance.

_-Product Analysis:_ Identification of best-selling items and inventory moving trends.


**Repository Structure**

_online_retail.csv.zip:_ The raw transactional dataset compressed in a ZIP format. It contains historical sales records including Invoice Numbers, Stock Codes, Descriptions, Quantities, Invoice Dates, Unit Prices, and Customer IDs.

_File1.pbix:_ The core Power BI Desktop file containing the data model, DAX measures, and interactive visual reporting layers.

_File1.pdf:_ A static PDF export of the finalized dashboard layout for quick viewing and presentations.


**Getting Started**

**Prerequisites**
To view or modify the interactive dashboard, you will need:
-Power BI Desktop (Free download)
-An application to extract .zip files (like WinRAR or 7-Zip)

**Installation & Setup**

1.Clone the Repository: git clone https://github.com/rsouparnika/Online-Retail-Performance-Dashboard.git

2.Extract the Data: Extract the online_retail.csv file from the online_retail.csv.zip archive.

3.Open the Dashboard: Launch Power BI Desktop and open File1.pbix.

4.Fix Data Source Paths (If Required):

If the visuals do not load automatically due to file path changes, go to Home > Transform Data > Data Source Settings in Power BI and point the source to your locally extracted online_retail.csv file.


**Dashboard Preview**

If you want a quick glance at the layout and insights without installing Power BI, you can open the File1.pdf file directly in your browser or any PDF viewer.
